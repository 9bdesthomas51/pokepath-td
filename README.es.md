# Pokepath TD

**Idioma:** Español | [English](README.md) | [Français](README.fr.md)

Pokepath TD es una experiencia de defensa de torres inspirada en Pokemon que puedes jugar al instante en tu navegador.

## Sitio web
- Juega ahora: https://pokepath-td.org/

## Ventajas
- Sin cuenta, juega al instante
- Experiencia limpia y rapida
- Herramientas de datos de Pokepath para comparar torres y planificar equipos
- Lista de niveles y base de datos de Pokemon para preparar la estrategia

## Que puedes hacer
- Entrar al juego de inmediato sin iniciar sesion
- Explorar estadisticas, habilidades y rutas de evolucion en la base de datos
- Comparar niveles y planificar tu mejor equipo antes de jugar

## Comunidad y comentarios
Si tienes ideas o comentarios, abre un issue o contacta desde el sitio web.
