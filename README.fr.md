# Pokepath TD

**Langue:** Français | [English](README.md) | [Español](README.es.md)

Pokepath TD est une experience de tower defense inspiree de Pokemon, jouable instantanement dans votre navigateur.

## Site web
- Jouer maintenant: https://pokepath-td.org/

## Points forts
- Aucun compte requis, jeu immediat
- Experience propre et rapide
- Outils de donnees Pokepath pour comparer les tours et planifier les equipes
- Tier list et base de donnees Pokemon pour preparer la strategie

## Ce que vous pouvez faire
- Lancer une partie sans connexion ni inscription
- Explorer les statistiques, capacites et evolutions dans la base de donnees
- Comparer les rangs et planifier votre meilleure equipe avant une partie

## Communaute et retours
Si vous avez des idees ou des retours, ouvrez une issue ou contactez-nous via le site.
